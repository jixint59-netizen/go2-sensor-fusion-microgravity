## GO2 four-material floor recognition

Archive: `go2_floor_material_recognition_v2_1_source.tar.gz`

The archive contains the complete text/source project, excluding large third-party GO2 OBJ mesh assets. The included `scripts/fetch_go2_assets.sh` script downloads those assets from the official Google DeepMind MuJoCo Menagerie repository.
